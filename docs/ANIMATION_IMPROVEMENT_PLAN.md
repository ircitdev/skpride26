## Инструменты

- **GSAP 3** — основная библиотека анимаций
- **ScrollTrigger** — scroll-based анимации
- **SplitText** — анимация текста
- **Flip** — layout анимации
- **Lenis** — smooth scroll (опционально)
- **Locomotive Scroll** — альтернатива (опционально)

---

## Метрики успеха

- **LCP** < 2.5s (Largest Contentful Paint)
- **FID** < 100ms (First Input Delay)
- **CLS** < 0.1 (Cumulative Layout Shift)
- **TTI** < 3.5s (Time to Interactive)
- **60 FPS** — все анимации без просадок
