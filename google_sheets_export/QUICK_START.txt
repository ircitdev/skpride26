📋 ШАГ 1: ЗАГРУЗКА В GOOGLE ТАБЛИЦЫ
════════════════════════════════════════════════════════════════

1. Откройте: https://docs.google.com/spreadsheets/
2. Создайте новую таблицу
3. Файл → Импортировать → Загрузить
4. Выберите файл: ВСЕ_ДАННЫЕ.csv
5. Настройки:
   ✓ Разделитель: Табуляция
   ✓ Кодировка: UTF-8
   ✓ Импортировать в: Новый лист

✏️ ШАГ 2: РЕДАКТИРОВАНИЕ
════════════════════════════════════════════════════════════════

Редактируйте колонки:
• Название - название услуги/тренера
• Подпись - короткое описание
• Краткое описание - текст для карточки
• Полное описание - расширенное описание
• Расписание - график работы
• Цена - стоимость услуг

⚠️ НЕ ТРОГАЙТЕ:
• ID
• Родитель
• Уровень
• Значение

📥 ШАГ 3: СКАЧИВАНИЕ
════════════════════════════════════════════════════════════════

1. Файл → Скачать → CSV (текущий лист)
2. Сохраните как: ВСЕ_ДАННЫЕ.csv
3. Замените файл в папке google_sheets_export

🔄 ШАГ 4: ИМПОРТ НА САЙТ
════════════════════════════════════════════════════════════════

1. Двойной клик по: import-from-google.ps1
2. Подтвердите импорт (Y)
3. Дождитесь завершения
4. Проверьте сайт
